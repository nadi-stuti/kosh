# 📅 Organise an Event

Running a river event is one of the most direct ways to take action — and one of the easiest to get wrong without a plan.

Every guide here is a **ready-to-use playbook**: a clear sequence of steps from initial planning through execution to post-event documentation. Pick the event type that matches what you want to do and follow the steps.

<Aside type="note" title="First time organising anything?">
Read [Coordinating a Group Drive](/get-involved/ground-work/intermediate/coordinating-a-group-drive/) in Ground Work first. It covers the core coordination skills all of these playbooks build on.
</Aside>
