## What this template will cover

A reusable post-event documentation template designed for any type of river event — cleanup, awareness walk, school visit, or monitoring drive.

**Planned content:**

- **Basic event record** — name, date, location (GPS + place name), organising group, number of participants, duration.
- **What was done** — activity summary, stretch covered, waste collected (type + approximate weight if applicable), observations recorded.
- **What went well** — a structured prompt for honest reflection on execution, participation, and community response.
- **What did not go well** — a blame-free prompt for capturing failures, near-misses, and logistical issues.
- **What we would do differently** — specific, actionable improvements for the next event.
- **Evidence and documentation** — checklist of photos, data sheets, and reports to attach.
- **Publishing your review on Nadikosh** — how to contribute your completed review to [Thinking Grounds → Community Spotlight](/thinking-grounds/) so other organisers can learn from your experience.

<Aside type="tip" title="Why reviews matter">
An honest event review shared on Nadikosh is one of the most valuable contributions you can make. It saves the next organiser from repeating your mistakes and builds a growing library of real-world river action knowledge.
</Aside>
