# Citizen Science Apps

<Badge text="Living section — changes frequently" variant="caution" />

Apps change fast. Guides in this section are among the most frequently updated in all of Get Involved.

Each app has its own dedicated guide with setup steps, key tasks, and a "last verified" date. If an app's interface has changed since the last update, leave a note in the Giscus thread on that page.

See [Pollution Academy → Citizen Science Apps](/pollution-academy/tools-and-tech/citizen-science-apps/) for a conceptual comparison of tools and their use cases.
