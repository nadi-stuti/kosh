## What this guide will cover

A practical, no-prior-GIS-experience guide to using free GIS tools for river field workers in India. This is not a GIS course — it covers only the tasks a field worker actually needs.

**Planned content:**

- **The only three tasks you need GIS for** — mapping your stretch, marking pollution hotspots, and exporting a map image for a report.
- **Tool choice** — Google My Maps (simplest, zero installation) vs QGIS (more powerful, free). When to use which.
- **Task: Create a stretch map on Google My Maps** — step-by-step from opening a new map to drawing your stretch, adding markers, and sharing a link.
- **Task: Import GPS coordinates from your phone** — how to export a KML/GPX file from a tracking app and load it into a map.
- **Task: Export a map image for a report** — how to generate a clean, labelled map screenshot suitable for a PDF report.
- **Introduction to QGIS for advanced users** — brief pointer to [Data & Research](/data-research/) for users ready to go deeper.
