# 🧭 Start Here

Before you pick a guide and head to the river, spend five minutes here.

Get Involved is built for people at very different stages — a college student doing their first cleanup, an NGO coordinator running a monitoring programme, or a citizen filing a pollution complaint for the first time. The guides are structured for all of them, but only if you know where to enter.

This short orientation will help you figure that out.

<Aside type="tip" title="Already know what you want to do?">
Skip this section and go straight to [Ground Work](/get-involved/ground-work/), [Digital Tools](/get-involved/digital-tools/), or [Organise an Event](/get-involved/organise-an-event/). Come back here if you feel lost.
</Aside>
