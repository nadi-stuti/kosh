## How to edit this wiki?

<CardGrid stagger>
	<Card title="Setup Github" icon="github">
		Read how to use github so that you can modify this wiki: [Github setup guide](/guides/github).
	</Card>
	<Card title="Change home layout" icon="document">
		Change contents in `src/content/docs/index.mdx` to modify home page
	</Card>
	<Card title="Add new content" icon="add-document">
		Add Markdown or MDX files to `src/content/docs` to create new pages.
	</Card>
	<Card title="Configure your site" icon="setting">
		Edit your `sidebar` and to add newly created pages in `astro.config.mjs`.
	</Card>
	<Card title="Read the docs" icon="open-book">
		Learn more in [the Starlight Docs](https://starlight.astro.build/).
	</Card>
</CardGrid>
